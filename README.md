# ↪︎ Sobre o projeto

Este projeto foi desenvolvido como uma **carta digital interativa**, criada especialmente para o **Dia dos Namorados**.

A página utiliza estilização e elementos visuais para transformar uma simples mensagem em uma experiência mais especial.

# ↪︎ Funcionalidades

✉ Página em formato de carta  
✉ Estilização com CSS  
✉ Interações com JavaScript  
✉ Funciona direto no navegador
